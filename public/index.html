Agent API
Let any AI agent store memories directly via HTTP
POST /api/store — Store a memory
Any agent can POST to this endpoint to store a memory on Shelby
// Store agent memory const response = await fetch('https://shelbyvault.vercel.app/api/store', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: 'agent-reasoning-001', type: 'reasoning', // reasoning | image | log | decision | research | checkpoint content: 'The agent concluded that...', agentId: 'gpt-4-turbo', tags: ['production', 'chain-of-thought'] }) }) const { blobName, txHash, explorerUrl } = await response.json()
Python Example
Works with any AI framework — LangChain, AutoGen, CrewAI...
import requests # After your agent makes a decision requests.post('https://shelbyvault.vercel.app/api/store', json={ 'name': f'decision-{agent_id}-{timestamp}', 'type': 'decision', 'content': agent.last_decision, 'agentId': agent_id, 'tags': ['autonomous', 'production'] })
